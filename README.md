# Análisis de Datos de la Industria del Cine

Este proyecto analiza un dataset de películas con información sobre presupuesto, ganancias, valoraciones y otros atributos, para explorar preguntas como la relación entre presupuesto y éxito en taquilla, y la fiabilidad de las valoraciones basadas en votos.

## Dataset
Datos tomados del archivo `tmdb_5000_movies.csv` con 4803 filas y 20 columnas.

## Cómo ejecutar
1. Clonar repositorio
2. Ejecutar `eda1.ipynb` en Jupyter Notebook

## Índice
- Importación del dataset
- Revisión de las dimensiones y primeras filas
- Revisión de los tipos de datos de las columnas
- Distribución inicial de las columnas
- Tratamiento de duplicados
- Tratamiento de valores nulos
- Corrección de incoherencias
- Análisis de datos
