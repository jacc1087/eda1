# Análisis Exploratorio de Datos de la Industria del Cine

Este proyecto analiza un dataset de películas con información sobre presupuesto, ingresos, valoraciones y otros atributos, para explorar preguntas como la relación entre presupuesto y éxito en taquilla, y la fiabilidad de las valoraciones basadas en votos.

## Dataset
Datos tomados del archivo `tmdb_5000_movies.csv` con 4803 filas y 20 columnas.

## Cómo ejecutar
1. Clonar repositorio
2. Instalar dependencias con `pip install -r requirements.txt`
3. Ejecutar `eda1.ipynb` en Jupyter Notebook

## Análisis
- Revisión de dimensiones y primeras filas
- Análisis de variables clave como presupuesto, ingresos, y puntuación
- Visualizaciones y conclusiones sobre el comportamiento del dataset
